# 🛒 Agentic SQL Data Analyst

Ask questions about any dataset in plain English — no SQL needed.

🔗 **Live Demo:** [sql-analyst-ai-agent.streamlit.app](https://sql-analyst-ai-agent.streamlit.app)

---

## 🚀 Features
- 💬 Natural language → SQL → Answer using LangChain SQL Agent
- 📤 Upload any CSV and query it instantly
- 🔍 See the exact SQL generated for every answer
- 📊 Auto stats and charts per dataset

## 🛠️ Tech Stack
LangChain · Groq (Llama-3.3 70B) · SQLite · Streamlit · Plotly · Pandas

---

## ⚙️ Run Locally

```bash
python database.py
streamlit run app.py
```

## 📊 Dataset
[Amazon Sales Dataset](https://www.kaggle.com/datasets/karkavelrajaj/amazon-sales-dataset) — 1,465 products with pricing, ratings, and reviews.
